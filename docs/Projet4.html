Projet 4 - API

Technologies : API | CRUD | JWT | UML | ScriptSQL | TypeScript

Le projet consiste à créer une API de gestion de produits et d'utilisateurs, avec des fonctionnalités CRUD (Create, Read, Update, Delete) sur une table de produits et une table d'utilisateurs. L'API doit inclure une authentification JWT, une gestion de rôles avec au moins deux rôles (USER/ADMIN), un UML de la base de données, une collection POSTMAN pour tester les requêtes, et un script SQL pour la création des tables et l'insertion de données par défaut.
