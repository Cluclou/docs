Projet 3 - Bubble my Tea

Technologies : Laravel | HTML | CSS | Git | MySQL

Le projet consiste à développer un site d'e-commerce pour commander des Bubble Tea en utilisant le framework Laravel. Le client souhaite des fonctionnalités telles que la page de connexion/inscription, un dashboard d'historique des commandes, une page de commande avec la possibilité de personnaliser les boissons, une page de profil utilisateur, et une page admin pour modifier la carte des boissons.
