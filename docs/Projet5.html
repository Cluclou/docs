Projet 5 - API on Docker

Technologies : API | DOCKER | Variables D'environnement | DockerHub

Le projet consiste à containeriser une API ainsi qu'une base de données en utilisant Docker. Les principales étapes comprennent la création de deux fichiers, un Dockerfile pour l'API et un docker-compose.yml. Les objectifs spécifiques incluent l'accessibilité de l'API depuis l'extérieur du conteneur, l'ouverture des ports nécessaires, la configuration des variables d'environnement pour l'API et la base de données, la liaison entre la base de données et l'API (LINK / depend_on), et la création de volumes pour la base de données. Enfin, le projet nécessite le push de l'image (conteneur) résultante sur une registry DockerHub privée préalablement créée.
