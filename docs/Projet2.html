Projet 2 - MyIRC

Technologies : TypeScript | ExpressJS | Git | Debian | Socket.io | MySQL | HTML | CSS | Trello

Le projet consiste à développer un IRC (Internet Relay Chat) en utilisant TypeScript, ExpressJS, Socket.io et MySQL. Le projet est divisé en plusieurs étapes, notamment la création du serveur ExpressJS, la gestion des messages avec Socket.io et HTML, et l'intégration d'une base de données MySQL pour la gestion des utilisateurs, de l'historique des messages, et des exports. Le projet doit être réalisé en groupe de 2 à 3 personnes, avec une durée d'une semaine.
